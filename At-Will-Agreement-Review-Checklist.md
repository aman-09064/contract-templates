**Purpose:**  
An At-Will Agreement allows either employer or employee to terminate the employment relationship at any time, for any reason, without cause (unless restricted by law).

| Section | Key Review Points | Status (✓/✗) | Notes |
|----------|------------------|---------------|-------|
| **Employment Nature** | Explicit statement that employment is at-will. |  |  |
| **Position & Duties** | Clear role description. |  |  |
| **Compensation & Benefits** | Salary, benefits, and deductions defined. |  |  |
| **Termination Clause** | Right to terminate at any time, without cause, by either party. |  |  |
| **Severance / Final Pay** | Conditions for final pay and benefits. |  |  |
| **Confidentiality & IP** | Protection of company information. |  |  |
| **Non-Discrimination** | Compliance with labor laws and equality statutes. |  |  |
| **Governing Law** | Applicable jurisdiction. |  |  |
