**Purpose:**  
A Service Contract defines the terms between a service provider and a client for specific deliverables, timelines, and payments. The goal is to ensure performance obligations, quality, and risk allocation are clear.

| Section | Key Review Points | Status (✓/✗) | Notes |
|----------|------------------|---------------|-------|
| **Parties & Purpose** | Full legal names, addresses, and business capacity of parties. |  |  |
| | Scope of services clearly defined with deliverables. |  |  |
| **Term & Termination** | Duration, renewal terms, and termination notice period. |  |  |
| | Termination for cause and convenience clauses. |  |  |
| **Payment Terms** | Fees, milestones, and payment schedule. |  |  |
| | Penalties or late payment provisions. |  |  |
| **Performance Standards** | Defined SLAs, KPIs, or benchmarks. |  |  |
| | Remedies for non-performance. |  |  |
| **Liability & Indemnity** | Limitation of liability caps. |  |  |
| | Indemnification clauses covering negligence or breach. |  |  |
| **Confidentiality** | NDA or confidentiality clause included. |  |  |
| **IP Rights** | Ownership of deliverables, licensing terms, and moral rights. |  |  |
| **Governing Law** | Jurisdiction and dispute resolution method. |  |  |
| **Force Majeure** | Coverage of events beyond control. |  |  |
