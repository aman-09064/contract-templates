
##  EMPLOYMENT CONTRACT REVIEW CHECKLIST

**Purpose:**  
An Employment Contract governs the relationship between employer and employee, outlining rights, obligations, benefits, and conditions of employment.

| Section | Key Review Points | Status (✓/✗) | Notes |
|----------|------------------|---------------|-------|
| **Employee Details** | Full name, designation, and reporting structure. |  |  |
| **Job Role & Duties** | Job title, role clarity, and performance metrics. |  |  |
| **Compensation** | Base salary, incentives, bonuses, and payroll terms. |  |  |
| **Benefits** | Leave, health, retirement, and insurance coverage. |  |  |
| **Probation & Termination** | Probation period, termination notice, and grounds for dismissal. |  |  |
| **Non-Compete & Non-Solicit** | Duration, scope, and enforceability. |  |  |
| **Confidentiality** | Post-employment confidentiality obligations. |  |  |
| **IP Ownership** | Employer ownership of employee-created work. |  |  |
| **Grievance & Arbitration** | Procedure for disputes. |  |  |
| **Governing Law** | Jurisdiction applicable. |  |  |
