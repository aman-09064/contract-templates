**Purpose:**  
A Freelancing Contract governs the relationship between a client and an independent contractor, ensuring clarity on deliverables, payment, ownership, and liabilities.

| Section | Key Review Points | Status (✓/✗) | Notes |
|----------|------------------|---------------|-------|
| **Scope of Work** | Detailed description of deliverables and timelines. |  |  |
| **Payment Terms** | Rate, milestones, method, and currency. |  |  |
| **Ownership & IP Rights** | Who owns the work product post-payment. |  |  |
| **Revisions & Rework** | Number of revisions included and cost of extra work. |  |  |
| **Confidentiality** | Obligation to protect client data. |  |  |
| **Termination** | Conditions for termination and payment upon cancellation. |  |  |
| **Indemnity & Liability** | Liability limits and indemnification. |  |  |
| **Non-Solicitation** | Restriction on working with client’s customers directly. |  |  |
| **Dispute Resolution** | Method (arbitration/mediation). |  |  |
| **Governing Law** | Jurisdiction and venue. |  |  |
