**Purpose:**  
A Sales Contract specifies the terms for the sale and purchase of goods or services, ensuring clarity on delivery, payment, and risk transfer.

| Section | Key Review Points | Status (✓/✗) | Notes |
|----------|------------------|---------------|-------|
| **Product Description** | Goods/services identified with specifications. |  |  |
| **Price & Payment** | Price, taxes, and payment terms defined. |  |  |
| **Delivery Terms** | Delivery schedule, method, and Incoterms (FOB, CIF, etc.). |  |  |
| **Inspection & Acceptance** | Procedures for product verification and acceptance. |  |  |
| **Warranty** | Duration, coverage, and exclusions. |  |  |
| **Title & Risk Transfer** | When ownership and risk pass to buyer. |  |  |
| **Returns & Refunds** | Process and conditions for returns. |  |  |
| **Liability & Indemnity** | Limitations and indemnification clauses. |  |  |
| **Force Majeure** | Definition and scope. |  |  |
| **Dispute Resolution** | Governing law and arbitration clauses. |  |  |
